# Axismundi Brand Assets

`axismundi-symbol.svg` is the canonical draft symbol for the Axismundi
project.

It is not release-locked yet. Final optical correction, small-size testing,
favicon derivation, screenshot integration, and plugin icon derivation should
happen after the block theme template skeleton is stable.

## Files

| File | Role |
|---|---|
| `axismundi-symbol.svg` | Canonical draft project symbol |
| `axismundi-symbol-fallback.svg` | Fallback symbol variant |
| `axismundi-symbol-glow.svg` | Glow / presentation variant |
