## SECTION A — Methodology

### Step-by-step pilot protocol

Per Phase 8.20 strategic guidance:

1. **Choose under-analyzed bounded context(s)** — must
   NOT be one of 4 already-classified contexts (admin-ui /
   plugin-dev / i18n / interactivity)
2. **Pre-register archetype prediction BEFORE detailed
   analysis** — explicit hypothesis with confidence rating
3. **Analyze actual closure profile** — constitutional
   element profile reading using v2 vocabulary
4. **Score prediction quality** — accuracy + insight gain +
   non-tautology check

### Pilot target selection

Available bounded contexts (NOT already archetype-classified):
- **block-authoring** (many chunks; 6h SOFT-mode +
  Doctrine 5 + Bridge Pattern)
- **theme-config** (many chunks; settings + styles +
  appearanceTools)
- **style-engine** (4 chunks; partial Computational-
  architectural per Phase 8.9 P1)
- **data-layer** (2 chunks; entity-resolution + persistence)
- **editor-customization** (3 chunks; block-filters +
  slotfills + editor-hooks)
- **site-building** (multiple chunks; Doctrine 5 +
  Resolution Surface)

Selected pilot targets: **2 contexts**
- **editor-customization** (HIGH-CONFIDENCE prediction
  expected — similar to admin-ui Governance-heavy)
- **data-layer** (LOW-CONFIDENCE prediction — genuinely
  uncertain; tests V2 under uncertainty)

This selection covers high-confidence + low-confidence cases,
testing V2 across prediction confidence range.

### Scoring methodology

Per prediction, score against actual analysis on 4
dimensions:

| dimension | scoring criterion |
|---|---|
| **D1 — Archetype match** | Does pre-registered archetype match best-fit actual archetype? (binary YES/NO/PARTIAL) |
| **D2 — Element density prediction** | Does pre-registered constitutional element profile match observed density? (high/moderate/low/no overlap) |
| **D3 — Absence prediction** | Does pre-registered absence pattern match observed absences? (correctly predicted / partially predicted / missed) |
| **D4 — Insight gain (non-tautology check)** | Does archetype prediction provide insight beyond what element-profile-reading alone provides? (YES/NO/PARTIAL) |

**Aggregate scoring**:
- Strong V2 prediction: 4/4 dimensions match
- Moderate V2 prediction: 3/4 dimensions match
- Weak V2 prediction: 2/4 or fewer
