## Use cases (v1.5+)

| Use case | Suggested mapping |
|---|---|
| Instance color tag (federated SNS) | Each instance assigned one static palette (Blue, Cyan, Green, etc.) |
| Tag chip variation | Tag string hashed to one of the 10 colored palettes |
| Status indicators (online/away/offline) | Green / Yellow / Grey 50 |
| Reaction colors | Red 70 (heart), Yellow 70 (star), Blue 70 (informative) |
| Code syntax highlighting | Static palettes provide a pre-curated rainbow |

When integrating, map the static palette hex value into a *new* sys token namespace (e.g. `--md-sys-color-instance-blue-bg`) — don't pollute the core sys layer.
