# @wordpress/undo-manager

A simple undo manager.

## Installation

Install the module

```bash
npm install @wordpress/undo-manager --save
```

## API

### createUndoManager

Creates an undo manager.

*Returns*

- `UndoManager< T >`: Undo manager.
