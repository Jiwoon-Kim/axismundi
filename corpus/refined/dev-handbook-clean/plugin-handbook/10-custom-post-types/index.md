# Custom Post Types

WordPress stores the Post Types in the `posts` table allowing developers to register Custom Post Types along the ones that already exist.

This chapter will show you how to [register Custom Post Types](registering-custom-post-types.md), how to [retrieve their content from the database, and how to render them to the public](working-with-custom-post-types.md).
