These articles explore the basic structure of the WordPress REST API.

[Pagination](03-using-the-rest-api/pagination.md): work with large collections of resources & control how many records you receive from the REST API

[Linking & Embedding](03-using-the-rest-api/linking-and-embedding.md): understand how to read and modify connections between different objects, and embed related resources such as author and media data in the REST API’s responses

[Discovery](03-using-the-rest-api/discovery.md): determine what REST API resources a site supports, and where they are located

[Authentication](03-using-the-rest-api/authentication.md): authorize your REST API requests so that you can create, update and delete your data

[Frequently Asked Questions](02-frequently-asked-questions.md): see some of the most frequent inquiries about the REST API and learn how to solve common problems
