# Features

Welcome to the Features documentation. In this chapter, you will learn about various features supported by the WordPress block theming system.

This chapter does not have a singular focus. Instead, it offers information on several different topics that will help you work with more advanced features, stepping beyond the basics that you learned in [Global Settings and Styles](../03-theme-json/index.md) and [Templates](../04-templates/index.md).

## Navigating this chapter

Use the following links to locate a topic within this chapter. Each article covers a specific feature related to building block themes, and you can explore each at your own pace.

- [**Block Style Variations**](block-style-variations.md)**:** Build custom block style variations (aka: block styles) for your users to select.
- [**Block Stylesheets**](block-stylesheets.md)**:** Register per-block stylesheets to compartmentalize your custom CSS and make your theme load faster.
- [**Block Variations**](block-variations.md)**:** Register variations on the default settings for a block and let your users add them via the interface.
