### P3-2 - `role="row"`

Decision: implement row wrappers.

Reason:

- BACKLOG #19 explicitly listed row structure.
- Row wrappers are generated by JS with `role="row"`.
- Lab CSS uses `.ax-date-benchmark__week { display: contents; }`, preserving
  the existing seven-column visual layout.

### P3-3 - `aria-multiselectable`

Decision: Option A.

The grid sets:

- `aria-multiselectable="false"` in single-date mode
- `aria-multiselectable="true"` in range mode

Reason:

- Range mode can expose both start and end selected cells through
  `aria-selected`.
- The full range a11y model remains out of scope, but this attribute makes the
  current bounded contract more honest than single-selection semantics.
