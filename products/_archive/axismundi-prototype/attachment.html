color-tokens-overview.png

Material Design 3의 컬러 토큰 시스템을 reference / system / component 세 레이어로 시각화한 다이어그램. 각 토큰이 어떻게 의미적 역할에 매핑되는지 색상 그룹별로 정리했습니다.

다이어그램 설명

이 다이어그램은 Axismundi 디자인 시스템에서 채택한 Material 3 컬러 토큰의 계층 구조를 보여줍니다. 가장 아래 layer는 --md-ref-palette-*의 raw color value (HCT 색공간 기반 톤 단계), 중간은 --md-sys-color-*의 의미적 역할 (primary, surface, on-surface 등), 맨 위는 컴포넌트 레벨 변수입니다.

핵심 통찰

  • 토큰을 단순 변수가 아닌 의미적 계층으로 보면 시스템 확장이 쉬워집니다.
  • 같은 ref token이 light / dark theme에서 다른 sys token으로 매핑됩니다.
  • 컴포넌트는 sys token만 참조 — ref은 직접 사용하지 않습니다.

참고 출처

본 다이어그램은 Material 3 컬러 시스템 공식 문서를 기반으로 한국어 커뮤니티 학습용으로 재구성한 것입니다. 출처를 명시하면 자유롭게 사용 가능합니다.
