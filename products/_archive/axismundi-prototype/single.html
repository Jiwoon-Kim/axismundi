Material 3 нҶ нҒ° мӢңмҠӨн…ң мІҳмқҢл¶Җн„° л№Ңл“ңн•ҳкё°

к№ҖлҜјмҲҳ
В· 2026л…„ 5мӣ” 8мқј В· 12л¶„ мқҪкё°

л””мһҗмқё мӢңмҠӨн…ңмқ„ мІҳмқҢл¶Җн„° л№Ңл“ңн•ңлӢӨлҠ” кұҙ ліҙнҶө *м»ҙнҸ¬л„ҢнҠё лқјмқҙлёҢлҹ¬лҰ¬*лҘј л– мҳ¬лҰ¬м§Җл§Ң, лҚ” к·јліём Ғмқё layerлҠ” нҶ нҒ° мӢңмҠӨн…ңмһ…лӢҲлӢӨ. мқҙ кёҖмқҖ Material 3мқҳ ref / sys / comp 3-tier нҶ нҒ° кө¬мЎ°лҘј CSS custom propertyлЎң мҳ®кёҙ кІҪн—ҳкіј, к·ё мң„м—җ light/dark schemeмқҙ мһҗлҸҷмңјлЎң syncлҗҳлҸ„лЎқ м„Өкі„н•ң нҢЁн„ҙмқ„ м •лҰ¬н•©лӢҲлӢӨ.

мқҙ мӢңмҠӨн…ңмқҖ мқјл°ҳм Ғмқё нҶ нҒ° мӢңмҠӨн…ң к°Җмқҙл“ңмҷҖ л‘җ к°Җм§Җ л©ҙм—җм„ң лӢӨлҰ…лӢҲлӢӨ вҖ” мІ«м§ё, лӘЁл“  мғүмқҙ tokenл§ҢмңјлЎң н‘ңнҳ„лҗҳкі  hex literalмқҙ 0к°ң (state layerк№Ңм§Җ нҸ¬н•Ён•ҙм„ң). л‘ҳм§ё, н•ңкёҖкіј мҳҒл¬ёмқҙ м„һмқё ліёл¬ё typographyлҘј мӢңмһ‘ лӢЁкі„л¶Җн„° кі л Өн•©лӢҲлӢӨ. мқҙ л‘җ кІ°м •мқҖ лӮҳмӨ‘м—җ ActivityPub л§ҲмқҙнҒ¬лЎңлё”лЎңк·ёлҘј л§Ңл“Ө л•Ң м»ӨлӢӨлһҖ leverageлЎң мһ‘лҸҷн–ҲмҠөлӢҲлӢӨ.

мҷң нҶ нҒ°мқёк°Җ?

к°ҖмһҘ лӢЁмҲңн•ң лӢө: мғүмқҙлӮҳ мӮ¬мқҙмҰҲлҘј н•ң лІҲ л°”кҝҖ л•Ң 100кө°лҚ°лҘј м•Ҳ л°”кҫёкё° мң„н•ҙм„ң. н•ҳм§Җл§Ң мқҙкұҙ н‘ңл©ҙм Ғмқё лӢөмқҙкі , лҚ” к№ҠмқҖ мқҙмң к°Җ мһҲмҠөлӢҲлӢӨ вҖ” нҶ нҒ°мқҖ мқҳлҜёлҘј мқёмҪ”л”©н•©лӢҲлӢӨ. #1976D2лҠ” к·ёлғҘ нҢҢлһҖмғүмқҙм§Җл§Ң, --md-sys-color-primaryлҠ” "мӮ¬мҡ©мһҗк°Җ к°ҖмһҘ мЈјлӘ©н•ҙм•ј н•  м•Ўм…ҳ"мқҙлқјлҠ” мқҳлҜёлҘј к°–мҠөлӢҲлӢӨ. мқҳлҜёк°Җ мһҲмңјл©ҙ light/darkм—җм„ң мһҗлҸҷмңјлЎң лӢӨлҘё hex к°’мқ„ л§Өн•‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

нҶ нҒ°мқҖ мӢңк°Ғмқҙ м•„лӢҲлқј мқҳлҜёлҘј лӢӨлЈ¬лӢӨ. мқҳлҜёк°Җ лӘ…нҷ•н•ҳл©ҙ мӢңк°ҒмқҖ л”°лқјмҳЁлӢӨ.

ліёмқёмқҳ л””мһҗмқё мӢңмҠӨн…ң мқјкё° мӨ‘

3-tier кө¬мЎ° вҖ” ref / sys / comp

M3 specмқҳ н•өмӢ¬мқҖ нҶ нҒ°мқ„ 3к°ң layerлЎң л¶„н• н•ң кІғмһ…лӢҲлӢӨ:

  1. Ref (reference) вҖ” мӢӨм ң hex к°’мқ„ к°–лҠ” мғү нҢ”л ҲнҠё. мҳҲ: --md-ref-palette-primary-40.
  2. Sys (system) вҖ” мқҳлҜём Ғ л§Өн•‘. мҳҲ: --md-sys-color-primaryк°Җ lightм—җм„  palette-primary-40, darkм—җм„  palette-primary-80мқ„ к°ҖлҰ¬нӮҙ.
  3. Comp (component) вҖ” м»ҙнҸ¬л„ҢнҠёлі„ token. мҳҲ: --md-comp-button-container-colorк°Җ --md-sys-color-primaryлҘј к°ҖлҰ¬нӮҙ.

мқҙ л¶„лҰ¬к°Җ мҷң мӨ‘мҡ”н•ңк°Җ? к°Ғ layerк°Җ лҸ…лҰҪм ҒмңјлЎң ліҖкІҪ к°ҖлҠҘн•ҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ. лёҢлһңл“ң мғүмқ„ л°”кҝҖ л•җ refл§Ң мҲҳм •. лӢӨнҒ¬ лӘЁл“ң hexлҘј лҜём„ёмЎ°м •н•  л•җ sysл§Ң мҲҳм •. лІ„нҠјмқҙ primary лҢҖмӢ  tertiaryлҘј м“°кІҢ н•  л•җ compл§Ң мҲҳм •. ліҖкІҪмқҳ blast radiusк°Җ layer м•Ҳм—җ к°ҮнһҷлӢҲлӢӨ.

Ref layer м •мқҳ

M3 baseline нҢ”л ҲнҠёлҠ” 5к°ң мғүмЎ° Г— 13к°ң нҶӨ(0-100) мңјлЎң мӢңмһ‘н•©лӢҲлӢӨ. н•ң лІҲм—җ лӢӨ м •мқҳн•ҳл©ҙ нҶ нҒ° нҢҢмқјмқҙ л¬ҙкұ°мӣҢм§ҖлӢҲ, мІҳмқҢм—” primary / secondary / tertiary / neutral / neutral-variant 5к°ңл§Ң: 

:root {
  /* Primary нҶӨлі„ hex вҖ” Material Theme Builderм—җм„ң м¶”м¶ң */
  --md-ref-palette-primary-0:   #000000;
  --md-ref-palette-primary-10:  #21005D;
  --md-ref-palette-primary-20:  #381E72;
  --md-ref-palette-primary-30:  #4F378B;
  --md-ref-palette-primary-40:  #6750A4;
  /* ... 50, 60, 70, 80, 90, 95, 99, 100 */
}

мқҙ лӢЁкі„м—җм„  *м–ҙл–Ө нҶӨмқҙ lightм—җ м“°мқјм§Җ, м–ҙл–Ө нҶӨмқҙ darkм—җ м“°мқјм§Җ* кІ°м •н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. refлҠ” scheme-agnosticмһ…лӢҲлӢӨ.

Sys layer вҖ” light/dark scheme л¶„кё°

Sys лӢЁкі„м—җм„ң мІҳмқҢмңјлЎң schemeмқҙ л“ұмһҘн•©лӢҲлӢӨ. M3 specмқҖ lightм—җм„  primaryк°Җ нҶӨ 40, darkм—җм„  нҶӨ 80мқҙлқјкі  м •мқҳн•©лӢҲлӢӨ. мқҙкұё CSSлЎң: 

:root,
:root[data-theme="light"] {
  --md-sys-color-primary:        var(--md-ref-palette-primary-40);
  --md-sys-color-on-primary:     var(--md-ref-palette-primary-100);
  --md-sys-color-primary-container:    var(--md-ref-palette-primary-90);
  --md-sys-color-on-primary-container: var(--md-ref-palette-primary-10);
  /* ... */
}

@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    --md-sys-color-primary:    var(--md-ref-palette-primary-80);
    --md-sys-color-on-primary: var(--md-ref-palette-primary-20);
    /* ... */
  }
}
:root[data-theme="dark"] {
  --md-sys-color-primary:    var(--md-ref-palette-primary-80);
  /* ... к°ҷмқҖ л§Өн•‘ лӘ…мӢңм Ғ м Ғмҡ© */
}

мқҙ нҢЁн„ҙмқҳ мһҘм җ вҖ” м»ҙнҸ¬л„ҢнҠёлҠ” н•ӯмғҒ var(--md-sys-color-primary)л§Ң м°ёмЎ°н•ҳл©ҙ лҗ©лӢҲлӢӨ. light/dark л¶„кё°лҘј м»ҙнҸ¬л„ҢнҠёк°Җ мӢ кІҪ м“°м§Җ м•Ҡм•„лҸ„ мһҗлҸҷмңјлЎң л§һлҠ” hexк°Җ л“Өм–ҙк°‘лӢҲлӢӨ.

State layer вҖ” color-mix()лЎң нҶ нҒ°л§Ң мӮ¬мҡ©

M3мқҳ hover / focus / pressed мғҒнғңлҠ” *color* мһҗмІҙлҘј л°”кҫёлҠ” кІҢ м•„лӢҲлқј, state layerлқјлҠ” л°ҳнҲ¬лӘ… мғүмқ„ мң„м—җ м–№мҠөлӢҲлӢӨ. specмғҒ hoverлҠ” 8% л¶ҲнҲ¬лӘ…, focusлҠ” 12%, pressedлҠ” 16% л¶ҲнҲ¬лӘ…н•ң on-{role} мғү.

к°ҖмһҘ лӢЁмҲңн•ң кө¬нҳ„мқҖ ::before pseudo-element + opacity мЎ°м Ҳ. н•ҳм§Җл§Ң color-mix() CSS н•ЁмҲҳлҘј м“°л©ҙ *backgroundлҘј м§Ғм ‘ н•©м„ұ*н•ҙм„ң лҚ” к№”лҒ”н•ҳкІҢ мІҳлҰ¬лҗ©лӢҲлӢӨ: 

.ax-button:hover {
  background-color: color-mix(
    in srgb,
    var(--md-sys-color-on-primary)
    calc(var(--md-sys-state-hover-state-layer-opacity) * 100%),
    var(--md-sys-color-primary)
  );
}

color-mix(in srgb, A 8%, B) = "AлҘј 8%, BлҘј 92% л№„мңЁлЎң н•©м„ұ". --md-sys-state-hover-state-layer-opacity: 0.08мңјлЎң м •мқҳн•ҙл‘җл©ҙ state layer л№„мңЁлҸ„ *нҶ нҒ°*мқҙ лҗ©лӢҲлӢӨ. м •л§җ лӘЁл“  кІҢ нҶ нҒ°л§ҢмңјлЎң н‘ңнҳ„лҗ©лӢҲлӢӨ.

н•ңкёҖ + мҳҒл¬ё mixed body вҖ” мІҳмқҢл¶Җн„° кі л Ө

н•ңкёҖ мӮ¬мқҙнҠём—җм„ң мҳҒл¬ё лӢЁм–ҙк°Җ к°‘мһҗкё° л“ұмһҘн•  л•Ң м–ҙл–»кІҢ мӨ„л°”кҝҲ лҗҳм–ҙм•ј н•ҳлӮҳмҡ”? кё°ліё CSSмқҳ word-break: normalмқҖ н•ңкёҖм—җ м Ғмҡ©н•ҳл©ҙ *кёҖмһҗ лӢЁмң„*лЎң лҒҠм–ҙ buggy. word-break: break-allмқҖ лӢЁм–ҙ мӨ‘к°„мқ„ лҒҠм–ҙлІ„л Ө лҚ” к°ҖлҸ…м„ұ л–Ём–ҙм§җ.

м •лӢөмқҖ word-break: keep-all + overflow-wrap: anywhere мЎ°н•©: 

body {
  word-break: keep-all;       /* н•ңкёҖ вҖ” лӢЁм–ҙ лӢЁмң„лЎң мӨ„л°”кҝҲ */
  overflow-wrap: anywhere;     /* кёҙ мҳҒлӢЁм–ҙ/URL вҖ” м•Ҳм „н•ҳкІҢ break */
  line-height: 1.6;            /* mixed-script line-height parity */
}

мқҙ мһ‘мқҖ 3 мӨ„мқҙ н•ңкёҖкіј мҳҒл¬ёмқҙ м„һмқё ліёл¬ём—җм„ң мһҗм—°мҠӨлҹ¬мҡҙ мӨ„л°”кҝҲмқ„ ліҙмһҘн•©лӢҲлӢӨ. мҳҒл¬ёл§Ң мһҲлҠ” мҳҒм—ӯм—җм„  break-word м •лҸ„лЎң м¶©л¶„н•ҳм§Җл§Ң, ліё мӮ¬мқҙнҠёмІҳлҹј WordPressлӮҳ ActivityPubк°Җ н•ңкёҖ мӮ¬мқҙм—җ л“ұмһҘн•ҳл©ҙ keep-all + anywhereк°Җ лӢөмһ…лӢҲлӢӨ.

л§әмқҢл§җ

нҶ нҒ° мӢңмҠӨн…ңмқҖ л””мһҗмқё мӢңмҠӨн…ңмқҳ *кё°мҙҲ*к°Җ м•„лӢҲлқј *мІҷм¶”*мһ…лӢҲлӢӨ. м»ҙнҸ¬л„ҢнҠёлҠ” нҶ нҒ° мң„м—җм„ң мһҗлқјкі , нҶ нҒ°мқҳ мқҳлҜёк°Җ лӘ…нҷ•н•ҳл©ҙ м»ҙнҸ¬л„ҢнҠёлҸ„ лӘ…нҷ•н•ҙм§‘лӢҲлӢӨ. M3 specмқ„ к·ёлҢҖлЎң л”°лқјк°Җл©ҙ 8н• мқҖ мҳімқҖ кІ°м •мһ…лӢҲлӢӨ вҖ” лӮҳлЁём§Җ 2н• мқҖ ліёмқё лҸ„л©”мқё (н•ңкёҖ, microblog л“ұ)мқ„ мң„н•ҙ carefulн•ҳкІҢ м¶”к°Җн•ҳл©ҙ лҗ©лӢҲлӢӨ.

лӢӨмқҢ кёҖм—җм„ңлҠ” мқҙ нҶ нҒ° мң„м—җ 33к°ң м»ҙнҸ¬л„ҢнҠёлҘј м–ҙл–»кІҢ л№Ңл“ңн–ҲлҠ”м§Җ, к·ёлҰ¬кі  к·ё кіјм •м—җм„ң л§ҲмЈјм№ң caveatл“Өмқ„ лӢӨлЈЁкІ мҠөлӢҲлӢӨ.
