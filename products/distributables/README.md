# Distributables

Production-grade, semver-versioned products. Stable contracts. Suitable for installation in real WordPress sites.

- `themes/` — published block themes (none yet)
- `plugins/` — published plugins (none yet — `hct-color-panel`, `typography-inspector` planned)

Unlike `reference-implementations/`, these have backward-compatibility commitments and changelogs. They consume the binding ontology but provide stable extension points to users.
