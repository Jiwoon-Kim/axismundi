# Distributable Plugins

Planned plugin lineup (v3.1+):

- `hct-color-panel/` — HCT (Hue/Chroma/Tone) color picker replacing WP hex picker
- `typography-inspector/` — M3 typescale role hierarchy panel
- `dynamic-palette/` — seed-based palette generation (M3 Material You algorithm)
- `theme-switcher/` — light/dark/variant switcher with persistence
