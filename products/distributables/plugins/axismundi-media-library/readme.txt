Promote WordPress attachments to independent media objects. Phase 0 scaffold.

== Description ==

Axismundi Media Library promotes the WordPress attachment into an independent,
publishable media object with its own owner, visibility, and — in later phases —
virtual folders, rights, sensitivity, saved references, and federation.

This 0.0.1 release is the **Phase 0 compatibility boundary**: a non-destructive
Core / Independent relationship-mode toggle and a read-only attachment
parent-relationship scan (migration preview). It changes no media behaviour yet.

The full specification lives in the plugin's docs/ directory (SPEC, SECURITY,
ROUTING, COMPATIBILITY, DATA-MODEL, PHASES). First public MVP = Phase 0 + Phase 1.

== Installation ==

1. Upload and activate Axismundi Media Library.
2. Open Settings > Media Library (Axismundi). Activation changes no media.
