# Axismundi Asset Licenses

This file records per-file licensing for non-code assets bundled with Axismundi.
Theme code and templates are licensed under GPL-3.0-or-later unless a more specific
notice applies.

## Fonts

Bundled font files under `fonts/` are distributed under the SIL Open Font License
1.1. Each font family directory preserves a verbatim `OFL.txt` and a `source.txt`
with source and processing notes.

| Family | Path | License |
|---|---|---|
| Roboto Flex | `fonts/roboto-flex/` | OFL 1.1 |
| Roboto Serif | `fonts/roboto-serif/` | OFL 1.1 |
| Roboto Mono | `fonts/roboto-mono/` | OFL 1.1 |

Format conversion (WOFF2) and unicode-range subsetting are permitted modifications
under OFL 1.1; reserved font names are unchanged.

Regional CJK families are not bundled here. The theme's CJK fallback slots may
be filled by optional language-specific plugins; without one, the slots resolve
to the operating system font.
