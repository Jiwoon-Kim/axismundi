# Omphalos Notices

Omphalos is a Twenty Twenty-Five child theme by Jiwoon Kim.

## Material Symbols

This theme bundles Material Symbols Outlined.

- Source: https://fonts.google.com/icons
- Upstream project: https://github.com/google/material-design-icons
- License: Apache License 2.0
- Bundled file: `assets/icons/material-symbols-outlined/material-symbols-outlined.woff2`
- License file: `assets/icons/material-symbols-outlined/LICENSE.txt`

The Material Symbols font is used as an icon font. Its glyph table and axes are
preserved so Material Symbols ligature names work in WordPress and editor
contexts.

## Fonts

This theme bundles Google Fonts families under the SIL Open Font License 1.1.
Each font directory preserves its `OFL.txt` and `source.txt`.

- Roboto Flex
- Roboto Serif
- Roboto Mono
- Noto Sans KR
- Noto Serif KR
