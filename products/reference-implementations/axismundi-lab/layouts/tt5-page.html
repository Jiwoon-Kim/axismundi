About the project

This page specimen keeps the TT5 page template simple: a featured image, a page title, and the full post content. It does not introduce a query loop, sidebar, or discovery tail.

The source featured image block does not force a 3:2 ratio. This lab placeholder uses a stable 16:9 slot only to make the static specimen visible while preserving the ontology note that the original block has no explicit aspect ratio.

For Pilot translation, this route should remain a quiet editorial page template rather than absorbing landing-page behavior from the no-title variant.
