# Checkbox - WordPress Mapping (v3.6.10 Phase 2)

## Status

No WordPress binding is implemented in v3.6.10.

## Theme Boundary

The Checkbox lab module is a reference implementation. It does not create:

```txt
theme.json controls
core block bridge mappings
editor sidebar controls
form submission handlers
custom blocks
plugin runtime
```

## Future Mapping Notes

Potential future WordPress use cases belong outside this cycle:

```txt
editor setting toggles
form plugin controls
custom block attributes
site option UIs
```

Those are plugin/editor territory, not a theme bridge change.
