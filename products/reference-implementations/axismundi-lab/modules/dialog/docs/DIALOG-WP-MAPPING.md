# Dialog WordPress Mapping

## v3.6.11 Status

No WordPress binding is implemented for Dialog in v3.6.11.

This module is a lab interaction runtime surface only.

## Future Routing

Potential future WordPress use cases must be routed explicitly:

```txt
theme-only static visual pattern:
  possible only for non-interactive markup

interactive modal authoring:
  plugin/custom-block territory if persistent schema or editor behavior is needed

front-end runtime binding:
  requires a separate binding strategy and accessibility review
```

No `theme.json`, Pilot bridge, fixture, or WordPress block editor code changes
are part of v3.6.11.
