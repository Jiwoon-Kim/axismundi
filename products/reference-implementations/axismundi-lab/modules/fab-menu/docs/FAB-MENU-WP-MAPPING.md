# FAB Menu WP Mapping

Component: FAB menu #5

Cycle: v3.6.13

## Theme Boundary

The theme may render static FAB menu markup in patterns or templates when the
actions are presentational examples.

## Plugin Boundary

Production behavior such as creating posts, uploading media, scheduling, editor
commands, or federation actions belongs to plugin/custom-block territory.

## Non-Goals

- No WordPress block style is registered.
- No PHP is changed.
- No Pilot bridge file is changed.
- No editor toolbar behavior is implemented.
