# Radio - WordPress Mapping (v3.6.10 Phase 2)

## Status

No WordPress binding is implemented in v3.6.10.

## Theme Boundary

The Radio lab module is a reference implementation. It does not create:

```txt
theme.json controls
core block bridge mappings
editor sidebar controls
form submission handlers
custom blocks
plugin runtime
```

## Future Mapping Notes

Potential WordPress uses, such as editor setting choices or custom block
attribute options, belong to plugin/editor work. They are not theme bridge
changes.
