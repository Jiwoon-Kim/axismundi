Ripple v2 Lab Module

Ripple v2 is an animated progressive enhancement above the static components.css §0 state-layer foundation. Stable authoring uses data-ax-ripple; the legacy allowlist remains only as transitional compatibility.

Main styleguide Ripple v2 audit Historical v1 audit
Static catalog note. Keyboard activation uses native focus-visible state-layer feedback; v3.5.6 ripple is pointer-only. Reduced-motion users should see no radial expansion.

§2 Bounded opt-in

Bounded ripple is the default for rectangular or clipped hosts.

§3 Unbounded opt-in

Unbounded ripple is explicit for icon-centric or floating action surfaces.
