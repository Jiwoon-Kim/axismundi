# Split Button WP Mapping

Component: Split button #7

Cycle: v3.6.13

## Theme Boundary

The theme can render split-button markup as a pattern where actions are inert
or handled by external integration.

## Plugin Boundary

The primary action and menu item behavior usually cross into plugin territory:
save, publish, export, scheduling, federation actions, and editor commands are
not theme behavior.

## Non-Goals

- No core/button semantic route change.
- No WordPress block style registration.
- No Pilot bridge change.
