# Tabs WordPress Mapping

Status: v3.6.8 Phase 2 lab implementation.

## Mapping

- Candidate WordPress surfaces: grouped content patterns, editor pattern previews, and future tabbed block concepts.
- Current v3.6.8 artifact: lab-only pattern page.
- No Pilot theme binding is introduced.

## Boundary

- This module does not create a custom block.
- This module does not reinterpret WP core/navigation or core/button semantics.
- Tabs remain a lab component until a future WordPress binding plan defines serialization and editor behavior.

## Future Notes

A WordPress binding should decide whether tab panels are block containers, pattern sections, or plugin-owned interactive markup before implementation.
