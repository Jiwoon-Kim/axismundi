# Toolbar WP Mapping

Component: Toolbar #8

Cycle: v3.6.13

## Theme Boundary

The theme can render toolbar markup as a public surface for grouped actions.

## Plugin Boundary

Actual commands such as editor formatting, saving, sharing, or federation
actions belong to plugin/editor integration. The module demonstrates stateful
controls only.

## Non-Goals

- No block editor toolbar integration.
- No WordPress PHP.
- No Pilot bridge change.
