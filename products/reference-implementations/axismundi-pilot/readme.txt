Axismundi Pilot is a v3.6.0 reference implementation for Axismundi's Material 3
WordPress binding.

== Description ==

This is a pilot theme, not a WordPress.org submission package.

It validates Axismundi's public surface in a real WordPress block theme:

* Wave 1 components minus Carousel.
* Material 3 token graph consumption.
* WordPress core block styling through theme-level block styles.
* Post body rendering through the prose contract.

== Installation ==

Use wp-env from the repository root:

1. Start Docker Desktop.
2. Run `wp-env start`.
3. Run `wp-env run cli wp theme activate axismundi-pilot`.
